# Social Media's Effects on Student Performance and Health
This project works to analyze student survey responses to predict the impact of social media use on academic performance.

## Progress
Project will be updated throughout December as work is transferred from student to personal account.

## Input file
The survey results found on Kaggle should be downlaoded and used for data_preprocessing.py
(https://www.kaggle.com/datasets/adilshamim8/social-media-addiction-vs-relationships)

## Run Program
data_preprocessing.py should be run prior to any other analysis files
